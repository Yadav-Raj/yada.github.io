How I made a Stock Trading Bot

A rough journey through creating a trading bot using Python, Jupyter, and Pandas

The Steps to Success

The first step always requires the user to set up the working environment and we do that using Jupyter Notebook and Python as they are both incredibly powerful yet comparatively simple. For this project we also use Yahoo Finance as our primary source of information and the Pandas package, often referred to as the Python Data Science Package. It allows the user to analyze data.

Then, create a function to read the opening and closing data of the stock market and write different if statements for a bearish and bullish market. An example from my program is also present in the picture above that represents the chunk of the logical code required for this task. We now use the Oanda API to connect our code to the real market and make automated trades

Results

The stock market was configured well and produced almost 2% profitable returns per day, which would be difficult to emulate as a human. The problem here arises with the market sentiment predictor, which no one can predict as well as humans. After all, it is our own sentiments that we are judging on and artificial intelligence cannot match to that. That is the truth of the stock market as well, a lot of it is numbers and financial ratios but we do have to account for market sentiments and how people feel about investing their money into the market. Besides that, the stock market bot performs really well.
